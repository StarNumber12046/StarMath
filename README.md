# StarMath
## Installing
to install the package do *pip install https://github.com/StarNumber12046/StarMath*
## How to use this package**
### Examples
#### Factorial N
``` py
import starmath
starmath.factorial(4) #number must be an int
```

#### Addition
``` py
import starmath
starmath.addition(4, 5) # you need two int
```
